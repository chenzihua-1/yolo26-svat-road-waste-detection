# Contributing Guidelines
Thank you for your interest in **YOLO26-SVAT-Road-Waste-Detection**!
This project provides the official implementation of the YOLO26-SVAT model for small-target road waste detection, as described in our paper. We welcome contributions that improve model performance, code stability, reproducibility, and documentation.

## How to Contribute
### 1. Report Issues
If you encounter bugs, errors, or unexpected results when running the code, please open an issue with:
- Detailed environment (OS, Python version, PyTorch version, CUDA version)
- Exact command to reproduce the problem
- Expected behavior and actual behavior
- Error logs or screenshots (if applicable)

### 2. Propose Features & Improvements
We welcome reasonable suggestions for:
- Improved small-object detection performance
- Training/inference speed optimization
- Better generalization to complex road scenes
- Clearer documentation and usage examples

### 3. Submit Code Changes (Pull Request)
To contribute code:
1. Fork this repository to your own account
2. Create a new branch from `main`
3. Make your changes and ensure they are relevant to **road waste detection**
4. Test your code to maintain reproducibility
5. Submit a pull request with a clear title and description

Note: Contributions should focus on **object detection**; content unrelated to waste detection will not be accepted.

### 4. Improve Documentation
You are welcome to fix typos, clarify ambiguous descriptions, or improve README and comments to help other researchers reproduce our results.
